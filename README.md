# Task1 Table Project

## Overview
This project is a simple HTML webpage that displays a table of employee or user information. The table contains details such as Name, Age, Date of Joining, Date of Birth (DOB), and Location. The project also includes a linked CSS file (`sample.css`) for styling purposes.

## Features
- Displays a table with multiple rows of data.
- Includes a caption and header row for clarity.
- Supports responsive design with a viewport meta tag.
- External CSS file can be used to customize the table's appearance.
